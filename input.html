User Input

One of the most important features within C++ is the ability to gather user input. Gathering user input could be used for example to create a game of madLibs. The collection of user input allows for interactive programs, dynamic behavior, as well as decision making within a program. Using a char input for example may allow for a selection menu for a program.

To gather user input in C++, the "cin" object is used. It allows the program to read data from the user, which can then be processed, manipulated, or displayed accordingly. Properly handling user input is crucial for writing robust, user-friendly programs. However, developers should also be cautious about input validation to avoid potential security vulnerabilities like buffer overflows or injection attacks.

#include <iostream>
#include <string>


int main() {
        string name;
        int age;


// Gathering user input for name
        cout << "Enter your name: ";
        getline(std::cin, name);


// Gathering user input for age
        cout << "Enter your age: ";
        cin >> age;


// Displaying user input
        cout << "Hello, " << name << "! You are " << age << " years old." << endl;


        return 0;
}
